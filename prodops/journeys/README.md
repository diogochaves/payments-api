# Jornadas

O Framework ProdOps possui cinco jornadas.

## Jornadas clássicas

| Jornada | Propósito |
|---|---|
| [Discovery](discovery/) | Exploração, experimentos, validação de hipóteses |
| [Delivery](delivery/) | Implementação, testes, deploy e promoção |
| [Operation](operation/) | Operação, incidentes, runbooks, postmortems |

## Jornadas transversais

| Jornada | Propósito |
|---|---|
| [Assessment](assessment/) | Avaliação contínua de riscos, oportunidades e planos |
| [Diligence](diligence/) | Observação contínua, gatilhos, verificações e coordenação |

Assessment e Diligence observam continuamente as demais jornadas. Não representam apenas documentação — representam comportamento ativo do Framework.

## Execution Mode

As jornadas podem ser executadas em dois modos:

- **Upstream** — exploração, baixo compromisso
- **Downstream** — entrega governada, critérios obrigatórios

→ [Execution Model](../execution-model/README.md)
